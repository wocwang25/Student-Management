                    KẾT NỐI BACKEND VỚI MSSQL
B0: cài requirements -> pip install -r requirements.txt
B1: Tạo login như tài khoản .env (tạo khác cũng được, nhưng mà nhớ đổi luôn trong env)
B2: Cấp quyền cho user 
    Security -> Login -> <user> -> General -> default database chọn cái QLSVNhom
                                -> User Mapping -> chọn QLSVNhom và set db_owner

B3: chạy thử file test_sqlserver -> py test_sqlserver.py 
B4: Mở code chạy thử -> py manage.py showmigrations -> chạy được thì ok
